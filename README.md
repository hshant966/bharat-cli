# Bharat CLI 🇮🇳

A powerful, high-performance CLI for Indian-specific workflows and **Ads4you** automation. 

Built with the speed and capability of a **Yodha**, Bharat CLI integrates with **Tandem Browser** for aggressive scraping and **Remotion** for programmatic video generation.

## Features

- **Namaste**: Daily status and focus summary.
- **Jobs**: Automated Indian Gov Job searching via Tandem Browser (aggregates FreeJobAlert, SarkariResult, etc.).
- **Ads**: One-click video generation for Ads4you (port 18790).
- **Automation**: Integration with OpenClaw profiles for background task management.

## Installation

```bash
git clone https://github.com/your-username/bharat-cli.git
cd bharat-cli
npm link
```

## Usage

```bash
# Personalized greeting
bharat namaste

# Search for government jobs
bharat jobs --daily

# Generate a promotional video
bharat ads --generate
```

## Environment Setup

Ensure you have your **Tandem API Token** in `~/.tandem/api-token`.
